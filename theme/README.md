## Installation

### Method 1: Upload ZIP
1. Go to Shopify Admin > Online Store > Themes
2. Click "Add theme" > "Upload zip file"
3. Select the theme.zip file
4. Click "Publish" to make it live

### Method 2: Manual Upload
1. Go to Shopify Admin > Online Store > Themes
2. Click "Add theme" > "Upload zip file"
3. Upload the entire `theme` folder contents

## Configuration

### 1. Add Your Logo
1. Go to Theme Settings > Logo
2. Upload your logo image (recommended: 200x50px)

### 2. Set Up Live Chat (Tidio)
1. Create account at tidio.com
2. Get your Public Key
3. Go to Theme Settings > Live Chat
4. Enter your Tidio Public Key

### 3. Add Products
Add the 6 products in Shopify Admin > Products with the descriptions provided in `/data/products.json`

### 4. Create Pages
Create the following pages in Shopify Admin > Pages:
- About Us (use template: page.about-us)
- Contact (use template: page.contact)
- FAQ (use template: page.faq)
- Privacy Policy (use template: page.privacy-policy)
- Terms & Conditions (use template: page.terms-conditions)
- Shipping & Delivery (use template: page.shipping-delivery)
- Returns & Refunds (use template: page.returns-refunds)

### 5. Set Up Payments
1. Go to Settings > Payments
2. Enable Shopify Payments (Stripe)
3. Enable PayPal
4. For testing, use Shopify's test mode

### 6. Create Blog
1. Go to Online Store > Blog posts
2. Create a blog called "News"
3. Add articles as needed

## Customization

### Colors
Edit in Theme Settings > Colors:
- Primary: #00FF7F (neon green)
- Background: #000000 (black)
- Text: #ffffff (white)

### Fonts
- Headings: Orbitron (Google Fonts)
- Body: Inter (Google Fonts)
